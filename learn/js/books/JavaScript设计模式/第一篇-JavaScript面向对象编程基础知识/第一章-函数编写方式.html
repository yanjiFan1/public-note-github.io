表单验证 用户名，邮箱，密码
第一种方案
function checkName() {}
function checkEmail() {}
function checkPassWord() {}
这种方案分析
缺点：定义了三个全局变量
第二种方案
var checkName = function checkName() {}
var checkEmail = function checkEmail() {}
var checkPassWord = function checkPassWord() {}
这种方案分析
这个相对于上面的方法来讲：在团队开发的时候，使用变量定义，减少覆盖或被覆盖的风险，一旦被覆盖，所有的功能都会失效,这个时候会轻易发现错误
第三种方案-用对象收编变量
var checkObject = {
  checkName : funtion() {},
  checkEmail : funtion() {},
  checkPassWord : funtion() {},
}
这种方案分析
这个方法可以使用对象的.语法使用方法
